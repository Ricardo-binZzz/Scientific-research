# Figure Factory

## Goal

Produce thesis-quality figures from research data, simulation outputs, or conceptual workflows.

## Supported Figure Types

- Process flow
- Structure schematic
- Comparison chart
- Trend plot
- Error or deviation plot
- Simulation result plot

## Figure Spec

- Figure purpose
- Source data
- Expected audience
- Axis labels or layout labels
- Legend rules
- Output size
- Output format
- Caption draft

## Figure Quality Rules

- Keep text readable at thesis print size.
- Keep color choices distinguishable in grayscale.
- Keep one figure focused on one message.
- Make the data source explicit in the figure metadata.
