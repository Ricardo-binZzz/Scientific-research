# Reading and Notes

## Goal

Convert papers into structured notes that can be used for writing without reopening the full PDF every time.

## Reading Sequence

1. Read abstract and conclusion.
2. Inspect figures and tables.
3. Read the method and results sections needed for the current task.
4. Extract only the claims that matter to the current chapter or figure.

## Note Structure

- What problem the paper solves
- What method or model it uses
- What data or experiment it relies on
- What result matters
- What limitation matters
- What can be reused

## Citation Discipline

- Every substantive claim in notes should point back to a source page, section, or figure.
- If a statement is interpretive, mark it as such.
- Never promote a note into manuscript text without tracing it to source material.

## Output Types

- Chapter outline fragments
- Literature review paragraphs
- Method comparison bullets
- Figure inspiration notes
- Experiment or simulation checkpoints
