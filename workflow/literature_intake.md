# Literature Intake

## Goal

Turn a raw search result into a trusted research note that can be reused in writing, figures, and discussion.

## Source Rules

- Start with a clearly defined question or gap.
- Search across at least two source types when possible: journal databases, preprint servers, conference proceedings, patents, or vendor/application notes.
- Record the exact query, date, source, and filter used for every search.

## Search Log Fields

- Question
- Keywords
- Query string
- Source
- Date
- Filters
- Result count
- Notes

## Download and Naming

- Save every paper under a stable project folder.
- Use a filename that includes year, first author, short topic, and source identifier if available.
- Keep the PDF, metadata, and summary card together.

## De-duplication

- Check title, DOI, authors, and year before importing.
- Keep one canonical copy per paper.
- Record alternate copies as references, not separate entries.

## Intake Gate

- Do not move a paper into the active library until it has been skimmed.
- Minimum skim: abstract, figures, conclusion, and any method section needed to judge usefulness.
- Only then create the summary card.

## Summary Card Rules

- Capture the problem, method, data, contribution, limits, and reuse value.
- Distinguish facts from interpretation.
- Mark directly reusable material for literature review, method comparison, or figure inspiration.
