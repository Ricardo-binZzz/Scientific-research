# Paper Summary

- Title: Finite element analysis of fixture stiffness
- Authors: Wang
- Source: Manufacturing Letters
- Year: 2022
- DOI: 10.1000/demo-stiffness

## Problem

Fixture stiffness affects machining stability and deformation control.

## Method

The paper compares several support layouts using finite element simulation.

## Data

Support position, stress, and displacement.

## Key Figures

Figure 2 boundary conditions; Figure 5 displacement comparison.

## Main Result

Higher support stiffness reduces peak displacement in the modeled region.

## Limitation

The model does not include experimental cutting-force variation.

## Reuse Value

Useful for explaining why simulation checks should include displacement and stress ranges.
